# Рекомендації для 100% відповідності макету Main.jpg

## Виконано в цій ітерації

- **Картки напрямків**: Підключені фото `direction-drones.png` та `direction-artillery.png` до всіх п’яти карток; посилання змінено на «ДІЗНАТИСЯ БІЛЬШЕ» / «LEARN MORE».
- **Hero**: Заголовок зроблено білим (як у макеті).
- **Секції**: Заголовки «НАПРЯМКИ» та «КОМАНДИРИ» — помаранчеві.
- **Рамки**: Картки напрямків і картки командирів мають тонку помаранчеву обводку.
- **Напрямки**: На фон карток додано grayscale (85%), при hover — повний колір.
- **Секція «Будуємо»**: Фон змінено на чорний (#000).

---

## Опційні доопрацювання (за бажанням)

1. **Помаранчева CTA-картка** — виконано: додано білу кнопку «ПРИЄДНАТИСЯ» (посилання на `#contact`), «ЗРОБИТИ ДОНАТ» залишено як текстове посилання знизу.

2. **Сітка командирів (4+5)**  
   У макеті: перший ряд — 4 картки, другий — 5 (останні дві по центру). Зараз використовується `grid-template-columns: repeat(3, 1fr)` (3 колонки). Для точної сітки 4+5 можна змінити на медіа-запиті з 4 колонками на великих екранах і окремим правилом для другого ряду (5 елементів по центру).

3. **Гострий прямокутник** — виконано: картки напрямків, командирів та помаранчева CTA мають `border-radius: 0`.

4. **Додаткові фото для напрямків**  
   Зараз два зображення чергуються на п’яти картках. Якщо з’являться окремі фото під кожен напрямок (дрони, артилерія, інженерія тощо), достатньо додати їх у `images/` і вказати в `style="background-image: url('images/...');"` на відповідному `.direction-card__bg`.

5. **CSP для шрифтів**  
   Якщо Bebas Neue не завантажується, у `<meta Content-Security-Policy>` потрібно дозволити `font-src https://fonts.gstatic.com` (окрім `'self' data:`).
